Rotation carousel

API

Inputs
width Define width of item in carousel type: string
default: 300px
height Define height of item in carousel type: string
default: 200px
depth Define depth of the radius carousel type: string
default: 510px
period Period of item change. Number value in ms or string (ex. '30s' or '3000ms') type: number | string
default: 3500 (3500ms)
direction Rotate direction 'forward' or 'reverse'

Output
rotationStart(index: number) Emit event whet panel start rotation with index of rotate item
rotationEnd(index: number) Emit event whet panel start rotation with index of rotate item

Methods
startRotate(index: number) Emit event whet panel start rotation with index of rotate item

<rotate-carousel>
   <ng-template rotateCarouselItem>
     <div style= "width: 100%; height: 100%; background-color: ...">
