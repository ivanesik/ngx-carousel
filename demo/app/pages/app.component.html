Rotation carousel

API

Inputs

width

Define width of item in carousel

type: string
default: 500px
height

Define height of item in carousel

type: string
default: 500px
depth

Define depth of the radius carousel

type: string
default: 600px
period

Period of item change. Number in ms or string (ex. '30s')

type: number | string
default: 3500 (3500ms)

Output

startRotate(index: number)

Emit event whet panel start rotation with index of rotate item
